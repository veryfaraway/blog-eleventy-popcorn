# 인물 카드 사용 가이드

블로그 포스트에서 배우나 유명인을 언급할 때 시각적인 인물 카드를 삽입할 수 있습니다.

## 기본 사용법

```markdown
{% person "이름", "역할/설명", "이미지URL", "링크URL", "IMDb ID" %}
```

## 인라인(미니) 링크 사용법 (추천: 본문 문장 안에 넣기)

본문의 텍스트 크기 수준으로 작게 표시되는 “이름 + 작은 프로필 이미지” 링크가 필요하면 `personInline`을 사용하세요.

```markdown
{% personInline "이름", "이미지URL", "링크URL", "IMDb ID" %}
```

## 파라미터 설명

### `person` 파라미터

1. **이름** (필수): 인물의 이름
2. **역할** (선택): 역할이나 간단한 설명 (예: "썬더볼트 로스 장군 역")
3. **이미지URL** (선택): 프로필 이미지 URL (없으면 플레이스홀더 표시)
4. **링크URL** (선택): 커스텀 링크 (위키피디아, 공식 사이트 등)
5. **IMDb ID** (선택): IMDb ID (예: "nm0000148")

### `personInline` 파라미터

1. **이름** (필수): 인물의 이름
2. **이미지URL** (선택): 프로필 이미지 URL (없으면 플레이스홀더 표시)
3. **링크URL** (선택): 커스텀 링크 (위키피디아, 공식 사이트 등)
4. **IMDb ID** (선택): IMDb ID (예: "nm0000148")

## 사용 예시

### 1. IMDb 링크 사용 (추천)

```markdown
{% person "해리슨 포드", "썬더볼트 로스 장군 역", "https://m.media-amazon.com/images/M/MV5BMTY4Mjg0NjIxOV5BMl5BanBnXkFtZTcwMTM2NTI3MQ@@._V1_UY317_CR10,0,214,317_AL_.jpg", "", "nm0000148" %}
```

### 1-1. 인라인(미니) 링크 사용 (본문 문장 안에 넣기)

```markdown
이번에 {% personInline "해리슨 포드", "https://m.media-amazon.com/images/M/MV5BMTY4Mjg0NjIxOV5BMl5BanBnXkFtZTcwMTM2NTI3MQ@@._V1_UY317_CR10,0,214,317_AL_.jpg", "", "nm0000148" %} 가 합류했습니다.
```

### 2. 커스텀 링크 사용

```markdown
{% person "봉준호", "영화감독", "https://example.com/image.jpg", "https://ko.wikipedia.org/wiki/봉준호", "" %}
```

### 3. 이미지만 사용 (링크 없음)

```markdown
{% person "김철수", "주연 배우", "https://example.com/image.jpg", "", "" %}
```

### 4. 최소한의 정보만 사용

```markdown
{% person "이영희", "", "", "", "" %}
```

## IMDb ID 찾는 방법

1. IMDb 웹사이트에서 배우/감독 검색
2. 프로필 페이지 URL 확인
   - 예: `https://www.imdb.com/name/nm0000148/`
   - ID는 `nm0000148` 부분

## 이미지 URL 찾는 방법

### IMDb 이미지 사용
1. IMDb 프로필 페이지에서 프로필 사진 우클릭
2. "이미지 주소 복사" 선택
3. URL 사용

### 기타 소스
- 위키피디아 공식 이미지
- 공식 사이트 프로필 이미지
- Cloudinary 등 이미지 호스팅 서비스

## 디자인 특징

- 컴팩트한 디자인 (기본 카드: 이미지 40x40px, 패딩 축소)
- 이름에 링크 포함 (클릭 시 IMDb 또는 커스텀 링크로 이동)
- 역할 정보는 이름 아래 작은 텍스트로 표시
- 호버 효과 (그림자 및 테두리 색상 변경)
- 반응형 디자인
- 외부 링크는 새 탭에서 열림
- 이미지 로드 실패 시 플레이스홀더 자동 표시
- `personInline`: 본문 텍스트 크기 수준의 작은 “아바타+이름” 링크 (이미지 20x20px)

## 실제 사용 예시

```markdown
---
title: "해리슨 포드가 MCU 합류 소식"
---

얼마 전에 윌리엄 허트가 사망했다는 안타까운 소식이 있었는데요, 
이번에 해리슨 포드가 윌리엄 허트를 대신하여 '썬더볼트' 로스 장군 역을 맡아 
MCU에 합류하게 되었습니다.

{% person "해리슨 포드", "썬더볼트 로스 장군 역", "https://m.media-amazon.com/images/M/MV5BMTY4Mjg0NjIxOV5BMl5BanBnXkFtZTcwMTM2NTI3MQ@@._V1_UY317_CR10,0,214,317_AL_.jpg", "", "nm0000148" %}

마블이 해리슨 포드 정도 되는 스타 배우를 캐스팅함에 따라...
```

## 팁

1. **이미지 최적화**: 가능하면 60x60px 정도의 작은 이미지 사용 (자동으로 리사이징됨)
2. **일관성**: 같은 인물이 여러 포스트에 등장하면 동일한 이미지 사용
3. **저작권**: 공식 프로모션 이미지나 퍼블릭 도메인 이미지 사용
4. **성능**: 이미지는 CDN을 통해 제공되는 것이 좋음
5. **컴팩트**: 역할 설명은 간결하게 (예: "썬더볼트 로스 장군 역")
6. **마크다운 여백**: 카드가 너무 커 보이면 카드 자체가 아니라, 카드 바깥의 문단(`<p>`) 마진(typography)이 원인일 수 있습니다. 이 경우 shortcode 출력이 블록 HTML로 처리되도록(앞 공백 없이) 설정되어 있어야 합니다.
